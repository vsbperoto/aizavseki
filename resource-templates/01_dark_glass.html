Installation

First, install the core library via your preferred package manager. We recommend using npm or yarn for the best experience.

$ npm install @aizavseki/sdk

Authentication

Initialize the client with your unique API key securely stored in your environment variables.

import { AiClient } from '@aizavseki/sdk';

const client = new AiClient({
  apiKey: process.env.AI_API_KEY,
  environment: 'production'
});
